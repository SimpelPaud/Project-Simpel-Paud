Tips Agar Pengisian Lancar

Koneksi Stabil

Gunakan koneksi internet yang stabil untuk menghindari data hilang saat proses pengisian.

Penamaan File Template

Saat mengunggah file Excel, pastikan nama file sesuai format: NAMA_SATUAN_NPSN.

Format Isian

Isi nomor HP/WA tanpa spasi (contoh: 081234567890) dan tahun berdiri dengan 4 digit (contoh: 2010).
